# User Backup Script Lab

## Description
This lab demonstrates a simple bash script that:
1. Greets the user
2. Creates a file with the username + 1100
3. Compresses the file using `tar` as a BACKUP
4. Shows the new backup filename with current date and time

2. Run the script:
